Amazon Unlocked Mobile Reviews - Sentiment Analysis 🚀
📝 Project Description
This project analyzes a large-scale dataset of 400,000+ unlocked mobile phone reviews from Amazon. The goal is to extract customer insights, brand performance, and perform Sentiment Analysis to understand user satisfaction across different mobile manufacturers.
📊 Key Highlights & Insights
Market Leader: Samsung has the highest volume of reviews and strong customer loyalty.
Overall Sentiment: Most reviews are highly positive, with 5-star ratings being the most frequent.
Word Cloud Insights: Words like "Battery", "Camera", and "Screen" are the most discussed topics among happy customers.
Value Brands: Identified brands that offer high quality at low prices, such as OtterBox and Huawei.
🧠 Technical Analysis
Sentiment Classification: Categorized reviews into Positive, Neutral, and Negative.
Correlation: Studied how Price affects Rating and user expectations.
Hardware Analysis: Identified common complaints in negative reviews (e.g., charging ports, battery life).
🛠️ Tech Stack
Python (Pandas, NumPy)
Visualizations (Matplotlib, Seaborn, WordCloud)
